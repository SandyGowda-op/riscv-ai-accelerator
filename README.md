# riscv-ai-accelerator
 Entry-level RTL project exploring AI acceleration through a pipelined RISC-V processor with a custom matrix multiplication unit.
This project represents my entry into RTL design, focusing on the intersection of chip architecture and AI acceleration. It demonstrates how hardware design can contribute to AI workloads through the implementation of a 5-stage pipelined RISC-V processor integrated with a custom matrix multiplication accelerator.
